# ADSA for DCSS 0.11+
Tesserium ADSA is a tool that aims to automatically downloads morgue files from various known sites accoding to a given username and analyzes then to give statistic data, and also some funny results.

Usage:
`./adsa -[akpv] --<username>`
+ a => akrasiac
+ k => kelbi
+ p => project357
+ v => verbose
